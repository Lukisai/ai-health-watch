# AI Health Watch

Protecting youth from unsafe AI mental-health advice.  
Live site: [https://ai-health.netlify.app](https://ai-health.netlify.app) (example)

## Features
- PII-safe report submission (/api/report via Netlify Functions)
- Cases & methodology pages
- Youth & educator guidance
- Auto-deploy via Netlify (PR previews)
